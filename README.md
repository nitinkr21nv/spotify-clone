# Spotify-Clone 

This is simple webpage which is clone of Spotify app made using HTML and CSS. 

## Features 
- Spotify-like UI design
- Sidebar navigation menu
- Music player layout
- Hover effects
- Clean and modern UI
- Responsive layout(basic)

## Technologies Used
- HTML
- CSS

## How to Run
1. Download or clone the repo
2. run index.html file
